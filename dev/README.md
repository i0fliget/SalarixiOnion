# Что такое builder.sh

Это инструмент, написанный на Bash, который при запуске автоматически собирает проект (./dev/build).


# Зависимости

NodeJS (pkg, typescript, tauri)
Golang
C# (.NET)
Python (pyinstaller)

# Как использовать builder.sh

1. Откройте терминал (CMD/PowerShell).
2. Перейдите в дерикторию проекта SalarixiOnion (желательно).
3. Запустите билдер (builder.sh): bash ./путь/к/билдеру.
4. Укажите дерикторию для билдера (если вы в корне проекта, просто напишите t).
5. Дайте согласие на удаление старых сборок.
6. Готово.
